##### 概述

记录常用的操作，包括发布文章，重装系统搭建


##### 发布文章

###### 步骤

- 新建文章 : hexo new "postName"
- 生成静态页面 : hexo g
- 部署至Github : hexo d
- 生成，部署到网页可执行 `release.sh`

###### 文章格式　
- 文章目录　blog/source/_posts
- 临时文章  blog/tmp
- 预览页面折叠: 文章中加入

```C
---
title: 翻墙之路   // 以这里的名称为主
date: 2018-09-29 11:03:05
tags:
		效率工具
---
.... 
//预览页面折叠
<!-- more -->
```
